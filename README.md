- **Features:** Facial Recognition, QR-based Medical Passport, Real-time GPS SOS, Live Doctor Guidance, WhatsApp Follow-up Reminders

- **Tech Stack:** Next.js, FastAPI, Sentence Transformers, spaCy, Tesseract OCR
